-Demarer l'application
1.Ouvrir l'invite de commande CMD
2.Se positionner dans le dossier 'Application' grace à l'invite de commande
3.Taper la commande 'php artisan serve' pour demarer l'application
4.Executer le script situer dans le dossier 'Base de donnee/Script.txt' dans votre SGBD
5.Ouvrir le navigateur internet et saisir l'adresse 'http://127.0.0.1:8000' dans l'URL

-Pour s'authentifier en tant qu'administrateur il suffit de taper 'http://127.0.0.1:8000/Admin' 
dans l'URL de votre navigateur
